## Getting Started
1. Clone repository.
2. Open project in Android Studio.
3. Build with Gradle.
   
## Tech stack:
- Android
- Kotlin, Coroutines, Flow
- MVVM, DI with Dagger-Hilt
- RoomDB, Coil
